# Plugin Documentation Corpus

Place plugin-specific Markdown files in this directory so MCP tool
`search_plugin_docs` can return concise excerpts instead of full manuals.

Recommended naming:

- `Serum.md`
- `FabFilter_Pro-Q_3.md`
- `<plugin_uid>.md`

Tips:

- Prefer short sections separated by blank lines.
- Include parameter IDs/names exactly as shown in the plugin UI.
- Note known quirks (e.g., bipolar ranges, hidden modulation routing).
