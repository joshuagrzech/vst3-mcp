Control Your VST Plugins with AI

Use Claude, Cursor, or Gemini to tweak parameters in plain English—"make it brighter", "reduce reverb", "find the cutoff knob"—while your plugin runs in your DAW.

View on GitHub Quick Start

How it works

1. DAW Insert AgentAudio Wrapper on a track.
2. Wrapper Load any VST3 child plugin and expose it to MCP.
3. AI Your coding agent sends realtime parameter updates.

Features

One wrapper, any VST3 Insert a single plugin; host Serum, FabFilter, compressors, synths—anything.
Natural-language parameter search Ask "make brighter", "less harsh", "reduce reverb"—find the right knob by intent.
Realtime updates while audio plays Parameter changes apply on the audio thread; no need for .vstpreset files.
One-line install Add MCP support to Cursor, Claude Code, or Gemini CLI in seconds.

Supported agents

Cursor Claude Code Gemini CLI

Quick Start

Build the wrapper, install into your DAW plugin path, then add MCP to your coding agent:

cargo build --release --manifest-path crates/agentaudio-wrapper-vst3/Cargo.toml
./scripts/build-and-install-vst3.sh release ~/.vst3

cargo run --bin agentaudio-mcp -- install

Start the router daemon, then insert AgentAudio Wrapper in your DAW (Ardour, REAPER, Bitwig).
