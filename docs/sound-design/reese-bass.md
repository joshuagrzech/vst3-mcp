# Reese bass recipe

Start with two detuned saw oscillators and keep both centered in mono for low-end stability.
Use a gentle unison spread only in upper harmonics to avoid phase cancellation in the sub.

Route a low-pass filter and modulate cutoff with a slow LFO synced to tempo (for example 1/4
or 1/8). Keep modulation depth moderate so the bass moves without disappearing in the mix.

Saturate after filtering to add harmonics, then use a chorus or short dimension effect for width.
High-pass the side channel to keep sub frequencies mono.

For aggression, automate distortion drive in transitions and slightly increase filter resonance.
Control harshness with a post-distortion EQ dip around 2-4 kHz if needed.
